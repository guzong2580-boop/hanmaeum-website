# Landing Page Content Template

## 1. Hero Section

### 1.1 Target Definition

* 주요 타겟: 중소기업, 공공기관, 학교, 자영업자
* 산업/직군: 마케팅 담당자, 총무, 홍보 담당자
* 문제 상황: 인쇄물 제작 과정이 복잡하고 가격/품질이 불확실함

---

### 1.2 Core Promise (한 문장 약속)

```
기업과 기관이 인쇄·홍보물 제작 문제를 해결하고
빠르고 정확한 맞춤 제작으로 비용과 시간을 절감하도록 돕습니다.
```

---

### 1.3 Supporting Line

* 해결 방식 요약: 주문 → 견적 → 결제 → 제작
* 차별성 요약: 맞춤 견적 시스템 + 빠른 대응
* 실행 방식 요약: 온라인 주문 후 담당자 견적 제공

---

### 1.4 Primary CTA

* CTA 문구: 견적 요청하기
* 클릭 후 이동 방식: 주문 의뢰 페이지 이동

---

## 2. Problem Section (고객 문제 정의)

### 2.1 핵심 Pain 3가지

1. 인쇄 견적 비교가 어려움
2. 제작 과정이 번거롭고 느림
3. 품질 및 납기 불안정

---

### 2.2 기존 방식의 한계

* 현재 해결 방식: 전화 문의 / 방문 / 단순 주문
* 왜 효과가 낮은가: 견적 불투명, 응답 느림
* 반복되는 문제: 수정 반복, 납기 지연

---

### 2.3 문제의 비용

* 시간 손실: 견적 및 커뮤니케이션 증가
* 비용 손실: 재작업 비용 발생
* 기회비용: 마케팅 타이밍 손실
* 조직/개인 리스크: 일정 지연

---

## 3. Solution Section (해결 방식)

### 3.1 우리의 접근 방식

* 접근 원칙: 빠르고 정확한 견적
* 실행 방식: 주문 기반 맞춤 견적
* 구조적 차별성: 자동 주문 + 수동 견적 결합 구조

---

### 3.2 Solution 구조 설명

* 단계 1: 주문 의뢰 등록
* 단계 2: 담당자 검토 및 견적 산출
* 단계 3: 고객 확인
* 단계 4: 결제 후 제작 진행

---

### 3.3 핵심 차별 요소

1. 맞춤 견적 시스템
2. 원스톱 제작 서비스
3. 빠른 대응 및 납기

---

## 4. Value / Benefits Section (고객이 얻는 결과)

### 4.1 정량적 결과

### 4.2 정성적 변화

* 팀 차원: 업무 부담 감소
* 개인 차원: 빠른 의사결정

---

### 4.3 Before / After 비교

| Before | After |
| ------ | ----- |
| 견적 반복 문의 | 단일 요청으로 처리 |
| 제작 지연 | 빠른 제작 |
| 가격 불투명 | 명확한 견적 |
